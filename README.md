# Library System

This is a Spring Boot project for managing a library system. It allows users to manage books, authors, and genres.

## Implemented Features

- [x] Book management
- [x] Author management
- [x] Genre management
- [x] User management
- [x] Authentication with JWT Bearer Token
- [x] Search functionality

- [x] Autorenverwaltung
